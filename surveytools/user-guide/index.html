Overview

SurveyTools is a set of calculators used in survey/measurement workflows (e.g. interpolation, wedge, shore tank corrections, and unit conversions). Each module follows the same basic pattern: enter values, tap Calculate, review the result.

How to use the app (general)

  • Enter values using the numeric keypad. Select your preferred decimal separator in settings (dot or comma).
  • Calculate to run the selected module. If inputs are missing or invalid, the calculator will not produce a meaningful result.
  • Clear resets input fields (some screens may keep shared values to speed up repeated calculations).
  • Settings may include appearance and input options that affect the UI but not the math.
